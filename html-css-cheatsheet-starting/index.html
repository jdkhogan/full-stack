Gradient Borders in CSS

The basic way to create gradient borders is to use the border-image property.

.gradient-1 { border-width: 4px; border-style: solid; border-image: linear-gradient(to right, navy, orchid) 1; }

The linear gradient CSS function builds a gradient from a given set of properties. In plain English, this border is built as a gradient starting with the navy CSS color and transitioning to the CSS orchid color at right.


But a problem with this approach is that we cannot apply a border radius.

.gradient-2 { border-width: 4px; border-style: solid; border-image: linear-gradient(to right, navy, orchid) 1; border-radius: 50em [Not Functional.] }



To address this issue, we can instead use the background-clip property.

.gradient-3 { background: linear-gradient(white, white) padding-box, linear-gradient(to right, navy, orchid) border-box; border-radius: 25px; border: 4px solid transparent; }

Here the linear gradient CSS function is used to build two separate color blocks.

The first line ending in padding-box builds a white background for the content, while the second line ending in border-box builds our gradient.

Finally, we need to pass in a width for a transparent border.
